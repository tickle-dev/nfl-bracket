# Open Graph Image

You need to create an image file named `og-image.png` in this directory.

**Specifications:**
- Size: 1200x630 pixels
- Format: PNG or JPG
- Content: Should include:
  - "All Mixed up Family Bracket" title
  - NFL/Football imagery
  - Your branding/logo
  - Text like "Join our NFL Playoff Bracket Competition"

This image will be shown when someone shares your link on:
- SMS/iMessage
- Facebook
- Twitter
- WhatsApp
- Slack
- Discord
- etc.

You can create this using:
- Canva (canva.com)
- Figma
- Photoshop
- Any graphic design tool

Once created, place it in this `public` folder as `og-image.png`
